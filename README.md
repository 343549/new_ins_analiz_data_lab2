# Лабораторная работа №2: Кластеризация данных (K-средних и DBSCAN)

## Описание

Данный проект представляет собой лабораторную работу по кластеризации данных с использованием двух алгоритмов: **K-Means** и **DBSCAN**. Работа включает анализ как синтетических данных, так и реального датасета о пингвинах.

## Структура проекта

```
.
├── lab2_clustering.ipynb    # Основной ноутбук с кодом и анализом
├── penguins.csv             # Датасет с данными о пингвинах
└── README.md               # Документация проекта
```

- **scikit-learn** - алгоритмы машинного обучения (KMeans, DBSCAN, метрики)

## Содержание работы

### 1. Синтетические данные (make_blobs)

- Генерация синтетического датасета с 4 кластерами (1000 точек, 2 признака)
- Применение алгоритма **K-Means** для кластеризации
- Применение алгоритма **DBSCAN** для кластеризации
- Визуализация результатов обоих методов
- Подбор оптимального числа кластеров для K-Means:
  - Метод локтя (Elbow Method)
  - Коэффициент силуэта (Silhouette Score)

### 2. Реальный датасет (Penguins)

- Загрузка и предварительный анализ датасета `penguins.csv`
- Обработка пропущенных значений и аномалий
- Масштабирование данных (StandardScaler)
- Применение **K-Means** с подбором оптимального числа кластеров
- Применение **DBSCAN** с экспериментированием параметров:
  - `eps` (максимальное расстояние между точками одного кластера)
  - `min_samples` (минимальное число точек для формирования кластера)
- Сравнение результатов двух методов

## Основные результаты

### Синтетические данные

- **K-Means**: Успешно разделил данные на 4 кластера (силуэт ≈ 0.79)
- **DBSCAN**: Нашёл 3 кластера при eps=0.3, min_samples=5 (силуэт ≈ 0.74)
- Оптимальное число кластеров по методу локтя и силуэта: **k=4**

### Датасет Penguins

- **K-Means**: 
  - Оптимальное число кластеров: **k=3** (соответствует 3 видам пингвинов)
  - Коэффициент силуэта: **0.451**
  - Распределение: 131, 123, 86 точек по кластерам
  
- **DBSCAN**:
  - Найдено **4 кластера** при eps=0.5, min_samples=5
  - Коэффициент силуэта: **0.424**
  - Обнаружено **67 точек шума** (19.7% от всех данных)

### Выводы

- Для датасета penguins **K-Means показал лучшие результаты** по метрике силуэта и интерпретируемости
- K-Means лучше подходит для данных с известной структурой и ожидаемым числом кластеров
- DBSCAN полезен для автоматического определения числа кластеров и выявления выбросов, но требует тщательной настройки параметров

## Особенности реализации

- Все данные масштабируются перед применением алгоритмов (особенно важно для DBSCAN)
- Используются метрики качества: инерция и коэффициент силуэта
- Проводится визуализация результатов в 2D пространстве признаков
- Для DBSCAN выполняется систематический перебор параметров с визуализацией результатов

## Методы кластеризации

### K-Means

**Преимущества:**
- Требует задать число кластеров заранее
- Работает быстро даже на больших данных
- Все точки распределяются по кластерам

**Недостатки:**
- Не может находить кластеры произвольной формы
- Чувствителен к выбросам
- Предполагает сферические кластеры

### DBSCAN

**Преимущества:**
- Может находить кластеры произвольной формы
- Автоматически определяет число кластеров
- Устойчив к выбросам (помечает их как шум)

**Недостатки:**
- Требует подбора параметров eps и min_samples
- Может не работать хорошо при разных плотностях кластеров
- Медленнее на больших данных
